# Avito Analytics Tool

Локальный инструмент для быстрой работы с выгрузками статистики Avito.

## Что это

Продукт помогает загружать выгрузки объявлений из файлов, автоматически собирать сводную аналитику, сохранять исторические срезы и сравнивать периоды в одном интерфейсе.

Вместо ручной работы в таблицах пользователь получает:

- импорт выгрузок из `xlsx/xml`
- автоматический разбор данных по объявлениям
- сводку по направлениям
- расчет ключевых метрик
- сохранение срезов по периодам и городам
- историю срезов и графики
- редактирование сохраненных данных прямо в интерфейсе

## Чем продукт полезен

Инструмент сокращает время на рутинную аналитику и убирает необходимость вручную сводить данные в нескольких таблицах.

Основная польза:

- вся аналитика находится в одном месте
- ключевые показатели считаются автоматически
- меньше риска ошибок в формулах и ручном переносе данных
- проще сравнивать разные периоды и города
- можно быстро получить управленческую сводку по направлениям

## Для каких задач подходит

Продукт полезен для сценариев, где нужно регулярно анализировать рекламные и коммерческие показатели по объявлениям:

- показы
- просмотры
- контакты
- расходы
- продажи
- выручка
- ROI / ROAS
- производные конверсии

## Основные страницы

### `index.php`

Главная страница для импорта и просмотра текущей выгрузки.

Возможности:

- загрузка файла
- просмотр таблицы объявлений
- сводка по направлениям
- ручной ввод части бизнес-данных
- сохранение среза
- работа с загруженными файлами

### `history.php`

Страница истории сохраненных срезов.

Возможности:

- фильтрация по городу, направлению и метрике
- выбор нужных периодов
- сравнение периодов на графике
- редактирование сохраненных срезов

## Хранение данных

Продукт использует файловое хранение:

- импорты сохраняются локально
- исторические срезы сохраняются в `json`

Это позволяет быстро запустить инструмент без отдельной базы данных.

## Итог

Это простой и практичный аналитический инструмент, который превращает выгрузки Avito в удобную рабочую систему: импорт, сводка, история и графики в одном месте.
